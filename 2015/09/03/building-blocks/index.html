积木 - 享受搭建应用的乐趣

发表于 2015-09-03   |   分类于 Android   |  

积木是以 Izzy Leung 提供的 知乎日报 API 既知乎日报为数据展现内容；以 OptionsMenu（选项菜单）的形式为功能扩展入口；以 Design Support Library（依循 Material Design）为主导设计 UI。

How to use

那么，你该如何利用她呢？

好啦，其实我已经帮你完成了整体框架的搭建，你需要做的，仅仅是依据自己的实际需求，简单的替换下数据来源（API）然后改改 UI，就可以快速地完成一款全新应用的开发啦！

是不是，想想还有点小激动？那么赶紧前往 GitHub Star 与 Fork 吧！你的支持将成为我最大的动力与褒奖！

Features - 功能点

  • About - 关于页面
  • Pick Photo - 选择与裁剪照片
  • Pick Place - 选择位置（需安装 Google Play Services 与 翻墙 - WTF）
  • Settings - 详细设置（只实现 UI）
  • MaterialDrawer - 导航抽屉（只实现 UI）
  • Design Support Library 不完全使用
  • SwipeBackLayout - 右滑返回
  • 未完待续 ……
